Overview

This project analyzes changes in weather based on distance to the equator. This is accomplished by pulling data from over 500 randomly selected global cities from the OpenWeatherMap API. Data was read into a Pandas DataFrame, and visualized using MatPlotLib.


Relationships between weather and lattitude were determined according to four attributes: Cloudiness, Wind Speed, Maximum Daily Temperature, and Hummidity. The source data can be viewed on the "Data" page, plots of individual attributes and description of any relevant trends can be viewed under the appropriate "Plots" page.
